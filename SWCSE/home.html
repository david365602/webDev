Goals

Make it easier to find Cyber events in the valley without needing to subscribe to 30 different calendars. This calendar will hopefully unify all Cyber events, thus creating a stronger collaboration in tech. Additionally, it should assist project managers schedule events.

Recomendations

When attending any of these events, make sure to network with people. Don't run out the door because, the presentation sucked. Make your time worth going to the event and gain more knowledge from other security experts in the room. Make yourself known!
